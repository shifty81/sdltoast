# SDLDEV - Game Development Reference Files

## Overview
This folder houses text files for game development to use as a reference. It contains design notes, development conversations, and technical guidance related to building the Harvest Quest game with SDL2 and C++.

## Contents

- **sdlnewest1.rtf** - Core game development reference covering world/movement design, life sim loop, exploration & combat, and SDL2 implementation guidance.

## Purpose
These reference files serve as a knowledge base for the project, capturing design decisions, implementation strategies, and technical notes that developers can consult during development.

## Adding New Files
When adding new reference files to this folder:
1. Use descriptive file names that indicate the content
2. Include the date or version context where applicable
3. Keep files focused on a single topic or development area
