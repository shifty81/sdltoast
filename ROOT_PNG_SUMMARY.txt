================================================================================
TECHNICAL DETAILS
================================================================================

Detection Order:
  1. Check root directory for common tileset names
  2. Scan root directory for any .png files
  3. Sort PNG files alphabetically
  4. Load first as world tiles, second as characters
  5. Fall back to assets/ directories if root empty
  6. Fall back to colored rectangles if no PNGs found

Directory Reading:
  - Unix/Linux: Uses opendir() / readdir()
  - Windows: #ifdef for future Windows implementation
  - Files sorted for consistent ordering

Load Priority:
  Root (named) ‚Üí Root (alphabetical) ‚Üí assets/ ‚Üí Fallback
